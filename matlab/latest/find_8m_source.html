1%> \brief
2%> Return a list of scalar MATLAB strings containing the paths to
3%> all detected ``mpiexec`` binaries installed on the system and
4%> available in the environment path variable.
5%>
6%> \details
7%> The search strategy is to parse and search the paths in the environmental
8%> `PATH` variable of the runtime processor shell and return all mpiexec paths.
9%> Also, all ``mpiexec`` paths found via ``pm.sys.path.mpiexec.which(vendor)`` are returned.
10%> Additionally, if ``vendor`` is missing or is set to ``"Intel"``, also search the default
11%> installation directories of Intel MPI libraries on all operating systems.<br>
12%> Think of this functionality ``pm.sys.path.mpiexec.find(vendor)``
13%> as a more comprehensive of what ``pm.sys.path.mpiexec.which(vendor)`` does.
14%>
15%> \param[in] vendor : The input scalar MATLAB string, containing the MPI
16%> library vendor that should match the ``mpiexec`` binary.
17%> Possible values are:<br>
18%> - ``Intel``, representing the Intel MPI library.
19%> - ``MPICH``, representing the MPICH MPI library.
20%> - ``OpenMPI``, representing the OpenMPI library.
21%> (**optional**, default = ``""``)
22%>
23%> \return
24%> `pathList` : A list of scalar MATLAB strings containing the paths to
25%> all detected ``mpiexec`` binaries installed on the system.
26%> If the ``mpiexec`` is not found or does not match the specified ``vendor``,
27%> the output will be an empty list ``[]``.
28%>
29%> \interface{find}
30%> \code{.m}
31%>
32%> pathList = pm.sys.path.mpiexec.find()
33%> pathList = pm.sys.path.mpiexec.find(vendor)
34%>
35%> \endcode
36%> \final{find}
37%>

42function pathList = find(vendor)
43 pathList = [];
44 if 0 < nargin
45 vendorLower = lower(vendor);
46 if vendorLower == "impi"
47 vendorLower = "intel";
48 elseif vendorLower == "open-mpi" || vendorLower == "openrte"
49 vendorLower = "openmpi";
50 end
51 vendorList = vendorLower;
52 else
53 vendorList = ["Intel", "MPICH", "OpenMPI"];
54 vendorLower = "";
55 end
56 % The strategy is to search for any executable in the
57 % environmental paths whose name contains `mpiexec`.
58 paths = getenv("PATH");
59 paths = string(strsplit(paths, pathsep));
60 for path = paths
61 apps = pm.sys.path.list(path);
62 for icell = 1 : length(apps)
63 app = apps(icell);
64 if contains(app, "mpiexec") && isfile(app)
65 if 0 < nargin
66 name = lower(pm.sys.path.mpiexec.vendor(app));
67 if name ~= ""
68 if contains(name, vendorLower)
69 pathList = [pathList, app];
70 end
71 end
72 else
73 pathList = [pathList, app];
74 end
75 end
76 end
77 end
78 % Try more via `which()` and default installation paths.
79 for name = vendorList
80 path = pm.sys.path.mpiexec.which(name);
81 if path ~= ""
82 if isempty(pathList)
83 pathList = [pathList, path];
84 elseif sum(strcmp(pathList, path)) == 0
85 pathList = [pathList, path];
86 end
87 end
88 end
89 if nargin == 0 || contains(vendorLower, "intel")
90 possibilities = [ "C:\Program Files (x86)\Intel\oneAPI\mpi\latest\bin\mpiexec.exe" ...
91 , "C:\Program Files\Intel\oneAPI\mpi\latest\bin\mpiexec.exe" ...
92 , "/opt/intel/oneapi/mpi/latest/bin/mpiexec" ...
93 ];
94 for path = possibilities
95 if isfile(path)
96 if isempty(pathList)
97 pathList = [pathList, path];
98 elseif sum(contains(pathList, path)) == 0
99 pathList = [pathList, path];
100 end
101 end
102 end
103 end
104end
