function mex ( in  mexname,
in  config 
)

Return the vector of MATLAB strings containing the path(s) to the ParaMonte library MEX shared files.

Parameters
[in]mexname: The input scalar MATLAB string containing the (partial) name of a MEX file name without MEX extension.
(optional. If missing, all detected paths will be returned.)
[in]config: The input scalar (or vector of) MATLAB string(s) containing the (set of) keyword(s) to match in the identified MEX files paths.
(optional. If missing, all detected paths will be returned.)
Returns
mexpaths : The output vector of MATLAB strings containing the path(s) to the ParaMonte library MEX shared files.
The output mexpaths will be an empty list if no such paths are found.


Possible calling interfaces ⛓

mexpaths = pm.lib.path.mex()
mexpaths = pm.lib.path.mex([])
mexpaths = pm.lib.path.mex([], [])
mexpaths = pm.lib.path.mex(mexname)
mexpaths = pm.lib.path.mex([], config)
mexpaths = pm.lib.path.mex(mexname, [])
mexpaths = pm.lib.path.mex(mexname, config)
