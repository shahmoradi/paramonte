3 % Return ``true`` if and only if the input ``varval`` conforms with the
4 % specified input type ``vartype`` and the specified maximum size ``varsize``.
5 % Otherwise, return ``false``.
6 %
7 % Parameters
8 % ----------
9 %
10 % varval
11 %
12 % The input value to be converted to namelist-compatible value.
13 %
14 % vartype
15 %
16 % The input scalar MATLAB string containing the
17 % expected type of the value given by the input ``varval``.
18 %
19 % The following type-conformance rules apply:
20 %
21 % - if ``vartype`` is ``"string"``, then ``varval`` can be
22 % either a MATLAB ``string`` or ``char``.
23 %
24 % - if ``vartype`` is ``"integer"``, then ``varval`` can be
25 % either a MATLAB ``int8``, ``int16``, ``int32``, ``int64``,
26 % or a whole-number ``real`` value.
27 %
28 % - if ``vartype`` is ``"logical"``, then ``varval`` can be
29 % either a MATLAB ``int8``, ``int16``, ``int32``, ``int64``,
30 % a MATLAB ``real``, or a MATLAB ``logical`` value.
31 %
32 % - if ``vartype`` is ``"complex"``, then ``varval`` can be
33 % either a MATLAB ``int8``, ``int16``, ``int32``, ``int64``,
34 % a MATLAB ``real``, or a MATLAB ``complex`` value.
35 %
36 % - if ``vartype`` is ``"real"``, then ``varval`` can be
37 % either a MATLAB ``int8``, ``int16``, ``int32``, ``int64``,
38 % or a MATLAB ``real`` value.
39 %
40 % For all other object types, the type-conformance is verified by
41 % passing the input ``varval`` and ``vartype`` directly to the
42 % MATLAB intrinsic function ``isa()``
43 %
44 % varsize
45 %
46 % The input scalar MATLAB integer representing the
47 % maximum allowed size of the input value ``varval``.
48 % (**optional**. If missing, the maximum length of the
49 % input ``varval`` will not be checked.)
50 %
51 % Returns
52 % -------
53 %
54 % itis
55 %
56 % The output scalar MATLAB logical that is ``true`` if and only if
57 % the input ``varval`` conforms with the specified input type ``vartype``
58 % and the specified maximum size ``varsize``. Otherwise, it is ``false``.
