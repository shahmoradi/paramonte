2 %
3 % This is the Ellipse3 class for generating
4 % instances of 3-dimensional Ellipse3 plots
5 % based on the relevant MATLAB
6 % intrinsic functions.
7 %
8 % In the following documentation,
9 %
10 % 1. the variable ``ndim`` represents the number of ellipsoids in the input data.
11 % The value of ``ndim`` is inferred from the shapes
12 % of the input ``gramian`` and ``center`` arguments
13 % as ``max([2, size(gramian, 1), size(center, 1)])``.
14 %
15 % 2. the variable ``nell`` represents the number of ellipsoids in the input data.
16 % The value of ``nell`` is inferred from the shapes
17 % of the input ``gramian``, ``center``, ``zval``, and ``cval`` arguments
18 % as ``max([size(gramian, 3), size(center, 2), size(zval, 2), size(cval, 2)])``.
19 % If the above expression yields zero, then ``nell`` is set to ``75``.
20 %
21 % 3. the variable ``npnt`` represents the number of points used in visualizing each ellipsoid.
22 % The value of ``npnt`` is inferred from the shapes of the input arguments
23 % ``zval`` and ``czval`` as ``max(size(zval, 1), size(cval, 1))``.
24 % If the above expression yields zero or one, then ``npnt`` is set to ``100``.
25 %
26 % \note
27 %
28 % To generate a 2D plot of ellipsoids, simply execute the
29 % MATLAB command ``view(2)`` to change the camera view to 2D.
30 %
31 % Parameters
32 % ----------
33 %
34 % gramian
35 %
36 % The MATLAB (function handle returning an) object that can be either
37 %
38 % 1. an empty object (e.g., ``[]``), in which case,
39 % the value ``repmat(eye(ndim, ndim), 1, 1, nell)`` will be used.
40 %
41 % 2. a scalar, containing the value of the diagonal elements
42 % of the diagonal matrices representing the Gramian matrices
43 % of all ``nell`` ellipsoids in the input data.
44 %
45 % 3. a vector of shape ``[1, 1, nell]`` such that ``gramian(1, 1, iell)``
46 % represents the diagonal value to be used in constructing the Gramian
47 % matrix of the ellipsoid ``iell` in the input ``nell`` ellipsoid data.
48 %
49 % 4. a matrix of shape ``[ndim, ndim, 1]`` used in constructing the Gramian
50 % matrices of all ``nell`` ellipsoids in the input data.
51 %
52 % 5. a tensor of shape ``[ndim, ndim, nell]`` such that ``gramian(:, :, iell)``
53 % represents the Gramian matrix of the ellipsoid ``iell` in the input data.
54 %
55 % \note
56 %
57 % While it is possible to pass a data directly to this class,
58 % it is highly recommended to pass a function handle that returns
59 % such data when called, allowing the visualization data to be
60 % dynamically updated when needed.
61 %
62 % center
63 %
64 % The MATLAB (function handle returning an) object that can be either
65 %
66 % 1. an empty object (e.g., ``[]``), in which case,
67 % the value ``zeros(ndim, nell)`` will be used.
68 %
69 % 2. a scalar, containing the value to assign to all coordinates
70 % of the centers of all ``nell`` ellipsoids in the input data.
71 %
72 % 3. a vector of shape ``[1, nell]`` such that ``center(1, iell)``
73 % represents the value of all coordinates of the center of the
74 % ellipsoid ``iell` in the input ``nell`` ellipsoid data.
75 %
76 % 4. a vector of shape ``[ndim, 1]`` such that ``center(idim, 1)``
77 % represents the value of ``idim`` coordinate of the centers of
78 % all ``nell`` ellipsoids in the input data.
79 %
80 % 5. a matrix of shape ``[ndim, nell]`` such that ``center(:, iell)`` represents
81 % the coordinates of the center of the ellipsoid ``iell` in the input data.
82 %
83 % \note
84 %
85 % While it is possible to pass data directly to this class,
86 % it is highly recommended to pass a function handle that returns
87 % such data when called, allowing the visualization data to be
88 % dynamically updated when needed.
89 %
90 % zval
91 %
92 % The MATLAB (function handle returning an) object that can be either
93 %

95 % the value ``repmat(1 : nell, npnt, 1)`` will be used.
96 %
97 % 2. a scalar, containing the value to assign to the z-axis
98 % coordinates of all ``npnt`` points used in the visualization
99 % of ``nell`` ellipsoids in the input data.
100 %
101 % 3. a vector of shape ``[1, nell]`` such that ``zval(1, iell)``
102 % represents the value of z-axis coordinates of all ``npnt``
103 % points used in the visualization of the ellipsoid
104 % ``iell` in the input ``nell`` ellipsoid data.
105 %
106 % 4. a vector of shape ``[npnt, 1]`` such that ``zval(ipnt, 1)``
107 % contains the z-axis value of ``ipnt`` points in the representations
108 % of all ``nell`` ellipsoids in the input data.
109 %
110 % 5. a matrix of shape ``[npnt, nell]`` such that ``zval(:, iell)`` represents
111 % the z-axis coordinates of all ``npnt`` points used in the visualization of

777 for idim = 1 : size(self.dims, 1)
778 for iell = 1 : length(self.workspace.ellindex)

781 iset = (idim - 1) * length(fields) * length(self.workspace.ellindex);
782 bcrd(:, iset + icol : iset + icol + 1) = pm.geom.ell2.getBorder ( self.workspace.gramian(self.dims(idim, :), self.dims(idim, :), jell) ...

791 end

797 self.df = pm.data.DataFrame(bcrd);
