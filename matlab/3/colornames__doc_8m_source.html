1%% COLORNAMES Examples
2% The function <https://www.mathworks.com/matlabcentral/fileexchange/48155
3% |COLORNAMES|> matches the input RGB values or color names to the
4% closest colors from the selected palette. |COLORNAMES| returns the
5% same outputs whether matching names or RGB:
6%
7% [names,RGB] = colornames(palette,RGB)
8% [names,RGB] = colornames(palette,names)
9%
10% This document shows some examples of using |COLORNAMES| to match
11% RGB values or color names. The bonus functions are also explained in this
12% document, together with examples.
13%% Palette Descriptions
14% Palettes of named colors have been defined by various people and groups,
15% often intended for very different applications. |COLORNAMES| supports a
16% wide selection of common color palettes: a detailed list of the supported
17% palettes is printed in the command window by simply calling |COLORNAMES|
18% with no input arguments and no output arguments:
19colornames()
20%% Return Palette Names
21% To return a cell array of the supported palettes simply call
22% |COLORNAMES| with no input arguments and one output argument:
23palettes = colornames()
24%% Return All Color Names and RGB Values for One Palette
25% Simply call |COLORNAMES| with the name of the required palette:
26[clr,rgb] = colornames('MATLAB')
27%% Match Color Names
28% Each input name is matched to a color name from the requested palette: an
29% input name that does not match any of the color names will throw an error.
30% The matching is very flexible though, as COLORNAMES usually makes a match
31% regardless of spaces between words and character case, although for some
32% palettes space characters may be significant. Note that CamelCase always
33% signifies separate words (words all in one case are considered one word).
34%
35% The color names are input as char row vectors and may supplied either
36% within one cell array or as separate input arguments:
37[clr,rgb] = colornames('xkcd',{'red','green','blue'})
38[clr,rgb] = colornames('xkcd','eggshell','eggShell')
39%% Match Index Number
40% Palettes with a leading index number may be matched by just the number,
41% or just the name, or both together:
42colornames('CGA','9','LightBlue','9LightBlue')
43%% Match Initial Letter
44% Palettes Alphabet, MATLAB, and Natural also match the initial letter to
45% the color name (except for 'Black' which is matched by 'k'):
46colornames('MATLAB','c','m','y','k')
47%% Match RGB
48% Each input RGB triple is matched to the closest RGB triple from the
49% requested palette:
50[clr,rgb] = colornames('HTML4', [0,0.2,1;1,0.2,0])
51%% Match RGB, Selecting the Color Difference Metric
52% Input RGB values are matched using one of several standard, well defined
53% <https://en.wikipedia.org/wiki/Color_difference color difference>
54% metrics known as $\Delta E$ or _deltaE_. The default color difference is
55% "CIE94", which provides good matching for most palettes and colors. Other
56% deltaE calculations can be selected by using the third input argument:
57rgb = [0,0.5,1];
58colornames('HTML4',rgb,'CIEDE2000')
59colornames('HTML4',rgb,'CIE94') % default.
60colornames('HTML4',rgb,'CIE76') % i.e. CIELAB.
61colornames('HTML4',rgb,'DIN99') % better than CIELAB.
62colornames('HTML4',rgb,'CMCl:c')
63colornames('HTML4',rgb,'RGB')
64%% View the Color Difference Metrics in a Figure
65% The helper function |COLORNAMES_DELTAE| demonstrates how the different
66% deltaE metrics match the input RGB to the palette colors. Simply
67% select the palette, provide an Nx3 colormap and all deltaE metrics
68% are listed, with the matched colors displayed in the columns below:
69colornames_deltaE('HTML4',jet(16))
70%% View the Palette Colors in 2D
71% The helper function |COLORNAMES_VIEW| plots the palettes in a figure.
72% Drop-down menus select the palette, and also how the colors are sorted.
73% Click on any color to view its hex RGB value (may be approximate).
74colornames_view('dvips','Lab')
75%% View the Palette Colors in 3D
76% The helper function |COLORNAMES_CUBE| plots the palettes in a figure.
77% The <http://www.mathworks.com/help/matlab/creating_plots/data-cursor-displaying-data-values-interactively.html
78% data cursor> can be used to view the color names, by clicking on the nodes.
79% Drop-down menus select the palette, and the color space of the colorcube:
80colornames_cube('CSS','Lab')
81%% Unmatched Color Name Error
82% If the input color name cannot be matched then |COLORNAMES| will throw an
83% error, and displays color names that are similar to the input string/s:
84colornames('CSS', 'bleu', 'blanc', 'rouge')
