Detailed Description

The ParaMonte MATLAB package pm.vis contains a set of routines and data related to data visualization tasks.

Specifically, the main goal of the routines and classes of this package is to simplify and automate much of the complexities of massive data visualization tasks, such as those encountered in statistical sampling.

Note
For more information on the existing functionalities, see the documentations of the package members.
