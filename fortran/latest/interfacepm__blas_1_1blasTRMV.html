Detailed Description

Return the following matrix-vector products, using the vector x and triangular matrix A or its transpose:

\begin{equation} x\leftarrow A x x\leftarrow A^T x x\leftarrow A^H x \end{equation}

This generic interface offers a compile-time resolution to the BLAS strmv, dtrmv, ctrmv, ztrmv matrix multiplication routines.
See the documentation of reference BLAS library for the definition of the input arguments.
