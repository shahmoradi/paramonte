Detailed Description

This is the timerMPI_type class, containing attributes and static methods for setting up a timer based on the OpenMP intrinsic omp_get_wtime().

See the documentation of timerOMP_typer for the non-default constructor interface of this type.
See also the documentation details of pm_timer.
