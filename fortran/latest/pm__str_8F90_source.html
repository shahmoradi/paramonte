1803! > \brief
1804! > Wrap the input scalar string to within the specified width while
1805! > optionally respecting the initial indentation in all subsequent lines.<br>
1806! >
1807! > \param[in] str : The input scalar of type `character` of kind \SKALL that is to be wrapped.<br>
1808! > \param[in] prefix : The input scalar of the same type and kind as the input `str` of arbitrary length type parameter,
1809! > whose contents will be prefixed to each new line (including the first line) before an indentation occurs.<br>
1810! > Note that `prefix` does **not** count toward filling the specified `width` or `maxwidth` of each wrapped line.<br>
1811! > (**optional**, default = `SKC_""`)
1812! > \param[in] indent : The input scalar of the same type and kind as the input `str` of arbitrary length type parameter,
1813! > whose contents are the set of characters to search for at the beginning of the input `str` representing the indentation.<br>
1814! > If specified as a non-empty input argument, then all wrappings of `str` will be indented with as many instances of `indent`
1815! > that are found at the beginning of the input `str`.<br>
1816! > For example, the following choice of input arguments,
1817! > \code{.F90}
1818! > List = getStrWrappedList(str = "~+~+This is a test.", indent = "~+")
1819! > \endcode
1820! > will lead to prefixing each wrapped line in the output `List` with `~+~+`.<br>
1821! > (**optional**, default = `" "`. If `indent = ""`, then no indentation of the lines will occur.)
1822! > \param[in] break : The input scalar of the same type and kind as the input `str` of arbitrary length type parameter,
1823! > whose contents are the set of characters at which breaking the input `str` is allowed.<br>
1824! > Setting `break = ""` will lead to wrapping the input `str` at any arbitrary character
1825! > as soon as the length of a given line reaches the specified `width`.<br>
1826! > (**optional**, default = " ")
1827! > \param[in] newline : The input scalar of the same type and kind as the input `str` of arbitrary length type parameter.<br>
1828! > Upon encountering a pattern that matches `newline`, the pattern will be removed and a new line will be created
1829! > starting with `prefix//indent` followed by the rest of the `str` that appears immediately after `newline`.<br>
1830! > (**optional**, default = `new_line(SKC_" ")`)
1831! > \param[in] width : The input scalar of type `integer` of default kind \IK, representing the length of each line in the
1832! > output wrapping, **with the possibility of word overflows**.<br>
1833! !> If a word overflows the specified input line width,
1834! > it is allowed to continue in the same line until the next `break` character is encountered.<br>
1835! > (**optional**, default = `132_IK`)
1836! > \param[in] maxwidth : The input scalar of type `integer` of default kind \IK,
1837! > representing the maximum length of each line in `List`, beyond which no character can appear in any line.<br>
1838! > If a word overflows the specified input `maxwidth` line limit, it will be broken and continued on the next line.<br>
1839! > If `break = ""`, then the value of `maxwidth` becomes irrelevant because line wrapping will strictly occur at `width`.<br>
1840! > (**optional**, default = `huge(0_IK) / 2_IK`)
1841! >
1842! > \return
