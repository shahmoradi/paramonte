Detailed Description

Generate and return the natural logarithm of the Probability Density Function (PDF) of observing a random (set of) point(s) at a (set of) distance (s) as the \(\kappa\)th nearest neighbor(s) of a given reference point in an ndim dimensional Poisson Point Process with a specified rate.

See the documentation of pm_processPoisson for more information on the calculation procedure.

Parameters
[in]distance: The input scalar or array of the same shape as other array like arguments, of type real of kind any supported by the processor (e.g., RK, RK32, RK64, or RK128), containing the value(s) for which the output logPDF must be computed.
[in]kappa: The input scalar or array of the same shape as other array-like arguments, of the same type and kind as distance, containing the rank of the nearest neighbor which must be considered in the calculations.
(optional, default = 1.)
[in]ndim: The input scalar or array of the same shape as other array-valued arguments, of the same type and kind as distance, containing the number of dimensions of the PPP.
(optional, default = 1.)
[in]rate: The input scalar or array of the same shape as other array-valued arguments, containing the rate parameter ( \(c_d \times \lambda\)) of the Poisson Point Process where \(c_d\) is the volume of the ndim-dimensional unit ball.
This quantity can be readily computed from the intensity parameter via getPoisRate.
(optional, default = 1.)
Returns
logPDF : The output scalar or array of the same shape as any input array-like argument, of the same type and kind the input argument distance, containing the natural logarithm of the Probability Density Function(s) of the specified distance (s).
