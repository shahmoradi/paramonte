Detailed Description

Generate and return the differential rate of an ndim dimensional Poisson Point Process at the specified input set of \(k\) nearest neighbors of an arbitrary reference in the process.

The differential rate parameter of a PPP with respect to an arbitrary reference point can be defined at the locations of the occurrences of the nearest neighbors of the reference point.
Denote the distance to the \(k\)th nearest neighbor of the reference point by \(r_k\).
Then, the corresponding PPP differential rate parameter for the \(k\)th nearest neighbor can be defined as,

\begin{equation} \large lambda_k = \frac{1}{c_d \left( r_k^d - r_{k - 1}^d \right)} ~, \end{equation}

where \(d\) is the dimension of the PPP (i.e., ndim), \(c_d\) is the volume of the unit ndim-ball computed via the generic interface getLogVolUnitBall, and \(r_0 = 0\).
.
For more information, see the documentation of pm_procPois.

Parameters
[in]knndis: The input vector of the same type and kind as the output rate of size \(k\) containing the distances to the first \(k\) nearest neighbors of a reference point in the PPP, from the first neighbor to the \(k\)th neighbor.
By definition, the input knndis must be sorted in ascending order.
[in]ndim: The input scalar of,
  1. type integer of default kind IK, or
  2. the same type and kind as the output rate,
containing the number of dimensions of the Poisson Point Process.
Returns
rate : The output vector of the same size as the input knndis of,
  1. type real of kind any supported by the processor (e.g., RK, RK32, RK64, or RK128),
containing the local PPP rate computed at the corresponding input nearest neighbors whose distances to the reference point are given by the input knndis.
