Return the average nearest-neighbor distance in dimension ndim with the rate density logDensity.

Parameters
[in]ndim: The number of dimensions.
[in]ndInverse: The inverse of ndim (i.e., 1._RK / ndim).
[in]logDensity: The natural logarithm of the target density.
[in]logVolUnitBall: The log-volume of unit ndim-ball.
Returns
logExpectedMinPairDist : A Symmetric matrix of shape (np,np) containing the pairwise distances-squared of the points from each other.
Remarks
The input logVolUnitBall can be computed via pm_ellipsoid::getLogVolUnitBall() and passed to this routine.
