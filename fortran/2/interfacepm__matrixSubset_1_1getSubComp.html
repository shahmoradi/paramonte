pm_matrixSubset::getSubComp Interface Reference

Detailed Description

Generate and return the objects representing the complementary subset of the input matrix subset sub with respect to the main diagonal of the matrix.

Parameters
[in]sub: The input scalar (or array of arbitrary rank and shape) of,
  1. type upp_type,
  2. type low_type,
  3. type dia_type,
  4. type uppDia_type,
  5. type lowDia_type,
  6. type uppLow_type,
  7. type uppLowDia_type,
Returns
subComp : The output object of the same rank and shape as the input subset sub whose type represents the matrix subset corresponding to the complementary subset of the input sub with respect to the main diagonal of the matrix.
  1. If sub is of type upp_type, then the output subComp is of type lowDia_type.
  2. If sub is of type low_type, then the output subComp is of type uppDia_type.
  3. If sub is of type dia_type, then the output subComp is of type uppLow_type.
  4. If sub is of type uppDia_type, then the output subComp is of type low_type.
  5. If sub is of type lowDia_type, then the output subComp is of type upp_type.
  6. If sub is of type uppLow_type, then the output subComp is of type dia_type.
  7. If sub is of type uppLowDia_type, then the output subComp is of type nothing_type.
