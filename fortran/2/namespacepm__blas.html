Detailed Description

This module contains a set of generic interfaces to the BLAS routines.

The BLAS generic interfaces of this module facilitate runtime dispatch of matrix algebra to the appropriate BLAS routines of an optimized BLAS library linked to the ParaMonte library.
The runtime dispatch occurs only if there is a corresponding routine (single or double precision) in the optimized BLAS library.
