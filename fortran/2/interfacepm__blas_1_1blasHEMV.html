Detailed Description

Return the result of the multiplication of a Hermitian matrix matA in Rectangular Default packing format with a vector/column-like matrix matB, added to a third vector/column-like matrix matC.

This generic interface offers a compile-time resolution to the BLAS chemv, zhemv Hermitian matrix multiplication routines.
See the documentation of reference BLAS library for the definition of the input arguments.
