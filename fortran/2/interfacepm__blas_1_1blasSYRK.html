Detailed Description

Return the Rank-K Update of a Real or Complex Symmetric or a Complex Hermitian matrix.

\begin{equation} C = \alpha A A^T + \beta C, C = \alpha A^T A + \beta C, \end{equation}

This generic interface offers a compile-time resolution to the BLAS ssyrk, dsyrk, csyrk, zsyrk matrix update routines.
See the documentation of reference BLAS library for the definition of the input arguments.
