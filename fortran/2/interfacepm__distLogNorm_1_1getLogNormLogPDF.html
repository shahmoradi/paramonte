pm_distLogNorm::getLogNormLogPDF Interface Reference

Generate the natural logarithm of probability density function (PDF) of the univariate Lognormal distribution. More...

Detailed Description

Generate the natural logarithm of probability density function (PDF) of the univariate Lognormal distribution.

Parameters
[in]x: The input positive-valued scalar or array of the same shape as other array-like arguments, of type real of kind any supported by the processor (e.g., RK, RK32, RK64, or RK128), representing the point(s) at which the PDF must be computed.
[in]mu: The input scalar or array of the same shape as other array-like arguments of the same type and kind as x representing the location parameter of the distribution.
(optional, default = 0)
[in]sigma: The input scalar of the same type and kind as x representing the inverse of the scale parameter of the distribution.
(optional, default = 1.)
Returns
logPDF : The output scalar or array of the same shape as the input array-like arguments, of the same type and kind as the input x representing the PDF of the Lognormal distribution at x.
