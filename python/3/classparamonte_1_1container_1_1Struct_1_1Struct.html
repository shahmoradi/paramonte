Detailed Description

This is a base class for generating objects that mimic the behavior of MATLAB struct.


This class is merely a convenience that is used internally within different routines of the ParaMonte Python library.


Possible calling interfaces ⛓

import paramonte as pm
struct = pm.container.Struct()
Note
This class primarily exist to facilitate bypassing the lack of struct (classes without methods) concept in Python.
