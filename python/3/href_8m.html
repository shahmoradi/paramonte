function href ( in  url)

Return an HTML-style decoration of the input URL if the ParaMonte MATLAB library is used in GUI, otherwise, return the input URL as is.

This functionality is important for properly displaying hyperlinks within the MATLAB command-prompt while avoiding the unnecessary HTML syntax clutter where it is not recognized.

Parameters
[in]url: The input scalar MATLAB string containing a bare url.
Returns
hlink : The output scalar MATLAB string containing the hyperlink (HTML-style-decorated) version of the input url.


Possible calling interfaces ⛓

hlink = pm.web.href(url)
