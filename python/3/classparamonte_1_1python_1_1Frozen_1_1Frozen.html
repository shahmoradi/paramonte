Detailed Description

This is a Python class for generating object instances whose attributes can be frozen upon request to avoid addition of further attributes to the object.

This class does not have a custom constructor.
The freezing of the object attributes is achieved by setting the internal property of the class instance _frozen to True, which is False by default.

Returns
fobj : The output scalar Python object of class pm.python.Frozen.
