Detailed Description

This is the ParaMonte Python main entrance module paramonte containing all functionalities available within the ParaMonte Python library.

The best and most recommended approach to using the library functionalities is to simply import the ParaMonte library as the main namespace as,

import paramonte as pm

and use the submodules and functionalities using the pm namespace.
For more information and example usage, see the documentation of the desired ParaMonte Python library functionality.

For more generic broad information on the ParaMonte Python library see the main page of the ParaMonte Python documentation.

Note
For more information on the existing functionalities, see the documentations of the package members.
