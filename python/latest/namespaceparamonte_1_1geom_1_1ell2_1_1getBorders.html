Return a Numpy matrix of Python doubles of shape [npnt, 2 * nell] (or [npnt, 3 * nell] if the input zval is present) containing the 2D (or 3D) coordinates of a set of points on the boundary of a set of 2D ellipsoids whose Gramian matrices and centers are specified as input argument.


Here nell refers to the number of ellipsoids which is equal to 2 * numpy.max(1, numpy.size(gramian, 2), numpy.size(center, 1)).

Parameters
[in]gramian: The input Numpy matrix of Python doubles of shape [2, 2, nell] containing the Gramian matrices of the nell 2D ellipsoids whose boundary points are to be returned.
(optional, default = numpy.reshape(numpy.eye(2), (2, 2, 1)))
[in]center: The input Numpy matrix of Python doubles of shape [2, nell] containing the 2D coordinates of the centers of the nell 2D ellipsoids whose boundary points are to be returned.
(optional, default = numpy.zeros((2, 1)))
[in]zval: The input scalar (or Numpy matrix of shape [npnt, nell] of) Python double(s) containing the z-axis coordinates of the points on the borders of the nell 2D ellipsoids.
If present, the specified value(s) will occupy the [1 : 2 : 3 * nell] columns of the output bcrd.
This argument is present for the sake of convenience and better performance of the algorithm to avoid further reallocations.
(optional. default = None. If missing or None, it will not be present in the output.)
[in]npnt: The input scalar Python whole number containing the number of points to return on the boundary of the target 2D ellipsoid.
(optional, default = 50 if numpy.size(zval, 0) > 1, otherwise, numpy.size(zval, 0).)
Returns
bcrd : The output Numpy matrix of Python doubles of shape [npnt, 2 * nell] (or [npnt, 3 * nell] if the input argument zval is present) containing the coordinates of a set of npnt points on the boundary of the target 2D ellipsoid.

Warning
The condition npnt == numpy.size(zval, 0) must hold for the corresponding input arguments.
